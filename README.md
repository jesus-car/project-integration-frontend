# 🏠 Roomly

**Roomly** es una aplicación mágica ✨ creada con **React** ⚛️, **Vite** ⚡, y **Tailwind CSS** 🎨. Este documento te guiará en cómo clonar el proyecto y configurarlo utilizando **Yarn** 🧶.

## 🛠️ Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

1. **Node.js** 📦 (versión 14 o superior)
2. **Yarn** 🧶 - Si no lo tienes instalado, puedes hacerlo con:
   ```bash
   npm install -g yarn
   ```
3. **VSCode** 📝 (recomendado)
   - Instala la extensión [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) para una mejor experiencia de desarrollo con Tailwind CSS.

## 🚀 Instalación

Sigue estos pasos para poner en marcha Roomly en tu máquina local:

### 1. 📥 Clona este repositorio
```bash
git clone https://github.com/usuario/roomly.git
```

### 2. 📂 Accede al directorio del proyecto
```bash
cd roomly
```

### 3. 📦 Instala las dependencias
```bash
yarn
```

### 4. 🏃‍♂️ Inicia el servidor de desarrollo
```bash
yarn dev
```

🌐 Dirígete a http://localhost:5173 para ver la aplicación en tu navegador.

## 🛠️ Configuración de Tailwind CSS

Tailwind CSS ya está configurado. Si deseas personalizar los estilos, modifica los siguientes archivos:

- 🎨 `tailwind.config.js` – Configura las rutas del contenido y ajusta el tema.
- 🔧 `postcss.config.js` – Administra los plugins de PostCSS.

💡 **Tip**: Con la extensión Tailwind CSS IntelliSense instalada en VSCode, obtendrás autocompletado y vista previa de clases de Tailwind mientras escribes tu código.

## 📜 Scripts Disponibles

- 🏃‍♀️ `yarn dev` – Inicia el servidor de desarrollo.
- 🏗️ `yarn build` – Compila el proyecto para producción.
- 👀 `yarn preview` – Previsualiza el proyecto después de compilar.
